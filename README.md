# Accounting Ledger

## Overview 

This application allows users to record and manage financial transactions

## Features
- [ ] Add deposit 

- [ ] Make payment

- [ ] Saves transactions to transactions.csv

- [ ] Prints all transactions

## Getting Started

- [ 1 ] Open the project in IntelliJ

- [ 2 ] Run the Main class
